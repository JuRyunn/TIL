### 클래스 (Class)
- 이상적인 세계에 존재하는 단 하나의 기준을 의미한다.
- 실제로는 존재하지 않는다.
- 가장 중요한 특성만 대략적으로 알려준다.
- 구체적인 수치가 없다.

```C#
int year= 2022;

switch(year) {
case 2021:
  Debug.Log("스파이더맨");
  break;
  
case  2022:
  Debug.Log("토르러브앤썬더");
  break;

default:
  Debug.Log("해당사항없음");
```

<br>



### 오브젝트 (Object)
- 자기 자신을 스스로 챙길 수 있다.
- 하나의 온전한 단위로 존재한다.
- 실존하는 세상에 있다.
- 하나의 원본에서 파생되어도 서로 구분이 가능하다.
