### var
- JavaScript에서 변수를 생성할 때 사용한다.

#### let
- 값을 수정할 수 있는 변수를 선언할 때 사용한다.

#### const
- 한번 지정하면 절대 변하지 않는 값인 상수를 선언할 때 사용한다.
- 새로운 값을 할당할 수 없다.
